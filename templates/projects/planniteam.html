PlanniTeam est une application web Django de planning d'équipe, consultable sur https://planniteam.herokuapp.com/. Le code source est disponible sur GitHub. Ce projet était le dernier de ma formation, projet libre pour lequel j'ai choisi de réaliser une application de planning d'équipe. Pour cela, j'ai analysé les besoins des utilisateurs, rédigé des spécifications fonctionnelles et techniques, planifié le projet avec un tableau Scrum sur Trello.com, réalisé l’application en Test Driven Development, et mis en ligne l'application sur Heroku.
